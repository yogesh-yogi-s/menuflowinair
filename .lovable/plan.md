# Phase 5 тАФ Public Menu Page

Wraps up the three-part roadmap by making each restaurant's menu publicly shareable at a stable URL, with a slug editor + QR code on the Profile page.

## What gets built

### 1. Public menu route тАФ `/m/:slug`
New file `src/routes/m.$slug.tsx`. Mobile-first (the user's preview is 400px wide), unauthenticated, SSR + SEO ready.

Layout:
- Header: restaurant avatar (if set), `restaurant_name`, `tagline`.
- Sticky horizontal category chips that scroll-spy to sections.
- One section per category. Items show name, description, price; unavailable items are dimmed with an "Unavailable" badge.
- Items with no `category_id` go under an "Other" section.
- Footer: "Powered by Lovable".

Data:
- Calls the existing `get_public_menu(_slug)` Postgres RPC (already in `public_menu_setup.sql`) via the browser supabase client тАФ no auth required, the RPC is `security definer` and grants execute to `anon`.
- Wrapped in a TanStack Query call with `staleTime: 60_000`.
- Loader uses `ensureQueryData` so HTML is rendered server-side.

SEO via `head()`:
- `title`: `${restaurant_name} тАФ Menu`
- `description`: tagline (or fallback)
- `og:title`, `og:description`, `og:image` (avatar URL when present)

Handles the "slug not found" case with `notFoundComponent`, and adds `errorComponent`.

### 2. Profile page тАФ slug + tagline + QR
Edit `src/routes/_authenticated/dashboard.profile.tsx`:
- New "Public menu" card with:
  - **Slug** input (with live availability check against `profiles.slug` тАФ debounced 400ms). Auto-suggests a slug from `restaurant_name` if empty.
  - **Tagline** input (short text, max 120 chars).
  - **Public URL** displayed as `https://{host}/m/{slug}` with Copy button + Open link.
  - **QR code** rendered client-side (`qrcode` package) with a "Download PNG" button.
- Uses a small new helper in `src/services/profile.ts`:
  - `updateMyProfile` extended to accept `slug` and `tagline`.
  - `isSlugAvailable(slug, excludeUserId)` тАФ `select id from profiles where slug = ?` returning boolean.

### 3. Sidebar nudge
`DashboardSidebar.tsx` gets a small "Share menu" link/button that opens the public URL in a new tab when the slug exists, otherwise links to the Profile page.

### 4. Dependency
- Add `qrcode` + `@types/qrcode` via `bun add`.
